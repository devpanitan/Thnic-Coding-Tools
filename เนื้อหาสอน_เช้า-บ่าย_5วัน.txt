หลักสูตร Introduction to Programming (ม.ปลาย) • 25–29 กรกฎาคม 2569
====================================================================

วันที่ 1 — 25 กรกฎาคม 2569  (ปูพื้น)
--------------------------------------------------------------------
ช่วงเช้า:
 • เปิดหลักสูตร – แนะนำ THNIC Academy, ประวัติอินเทอร์เน็ตไทย และโดเมนเนม .th (ใช้ simulator DNS)
 • สันทนาการ เกมละลายพฤติกรรม + แบ่งทีม ตั้งชื่อทีม
 • การคิดเชิงคำนวณ (Computational Thinking) 4 เสาหลัก

ช่วงบ่าย:
 • กิจกรรม Unplugged “หุ่นยนต์ทำแซนด์วิช” (โปรแกรม = ชุดคำสั่งที่ชัดเจน เรียงลำดับ)
 • รู้จัก C++ และเขียนโปรแกรมแรก Hello World
 • พื้นฐาน Syntax และการอ่าน error + สรุปวันแรก


วันที่ 2 — 26 กรกฎาคม 2569  (เน้น C++ #1)
--------------------------------------------------------------------
ช่วงเช้า:
 • โครงสร้างโปรแกรม (Structure code) และพื้นฐาน Syntax
 • Input และ Output (cin / cout)
 • ตัวแปร (Variable) และชนิดข้อมูล (Type)

ช่วงบ่าย:
 • ตัวดำเนินการ + − * / (Operators)
 • เงื่อนไข if / else (ใช้ simulator if-else)
 • เกมแข่งเขียนโค้ดสั้น ๆ เป็นทีม + ใบโจทย์ฝึก C++ เบื้องต้น


วันที่ 3 — 27 กรกฎาคม 2569  (เน้น C++ #2 / อาจหยุดพักเบรกให้น้อง ๆ ผ่อนคลาย)
--------------------------------------------------------------------
ช่วงเช้า:
 • กิจกรรมปูพื้น / วอร์มอัพสมองสบาย ๆ
 • Switch Case
 • การวนซ้ำ Loop (while, for) (ใช้ simulator Loop)

ช่วงบ่าย:
 • Array และการใช้ Loop ร่วมกับ Array (ใช้ simulator Array)
 • Mini-project เป็นทีม (เช่น เกมทายเลข / โปรแกรมคิดเลขอัตโนมัติ)


วันที่ 4 — 28 กรกฎาคม 2569  (โจทย์โอลิมปิก ระดับปานกลาง)
--------------------------------------------------------------------
ช่วงเช้า:
 • แนะนำแนวข้อสอบ Computer Olympic และเทคนิคถอดรหัส/วิเคราะห์โจทย์
 • ฝึกวางแผนและเขียนโค้ดแก้โจทย์ร่วมกัน (เป็นทีม)

ช่วงบ่าย:
 • ถอดรหัสและเขียนโค้ดข้อสอบ Computer Olympic ระดับปานกลาง (จับคู่ มีพี่เลี้ยงช่วย)


วันที่ 5 — 29 กรกฎาคม 2569  (โจทย์โอลิมปิก ระดับยาก + ปิดค่าย)
--------------------------------------------------------------------
ช่วงเช้า:

ช่วงบ่าย:

 • มอบเกียรติบัตร / ของรางวัล และปิดหลักสูตร
