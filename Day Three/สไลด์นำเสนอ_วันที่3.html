วันนี้เราจะได้เรียนอะไรบ้าง?

  • ทบทวน: เมื่อวานเขียนตัวแปร ตัวดำเนินการ และ if / else เป็นแล้ว
  • การวนซ้ำ (Loop) — ให้คอมทำงานซ้ำ ๆ ด้วย for
  • วนซ้ำอีกแบบด้วย while และรู้ว่าเมื่อไรใช้แบบไหน
  • อาเรย์ (Array) — เก็บข้อมูลหลายค่าในตัวแปรเดียว
  • ใช้ Loop + Array ด้วยกัน — วนอ่านข้อมูลทั้งชุด
ทำไมต้องมีการวนซ้ำ?

อยากพิมพ์เลข 1 ถึง 5...

ถ้าเขียนทีละบรรทัด — ก็พอไหว

แต่ถ้าให้พิมพ์ 1 ถึง 1000 ล่ะ? เขียน cout พันบรรทัด คงไม่ไหว

การวนซ้ำ (Loop) คือบอกคอมว่า “ทำซ้ำแบบนี้ N รอบ” — เขียนครั้งเดียว ทำงานกี่รอบก็ได้

// แบบเขียนเอง (เหนื่อย) cout << 1 << " "; cout << 2 << " "; cout << 3 << " "; cout << 4 << " "; cout << 5 << " "; // ...ถ้าถึง 1000 ล่ะ ?!
for loop — วนซ้ำตามจำนวนรอบ

เขียนครั้งเดียว ทำงาน 5 รอบ

#include <iostream> using namespace std; int main() { for (int i = 1; i <= 5; i++) { cout << i << " "; } return 0; }
int i = 1

① เริ่มต้น — สร้างตัวนับ เริ่มที่ 1

i <= 5

② เงื่อนไข — ทำต่อตราบที่ยังจริง

i++

③ เพิ่มค่า — จบรอบแล้วบวก 1

ผลลัพธ์: 1 2 3 4 5
ทำงานทีละรอบยังไง

คอมเดินวนแบบนี้

  • 1ตั้ง i = 1 → เช็ก 1 <= 5 จริง → พิมพ์ 1 → i++ เป็น 2
  • 2เช็ก 2 <= 5 จริง → พิมพ์ 2 → i เป็น 3
  • …วนแบบนี้จนพิมพ์ 3, 4, 5
  • 6เช็ก 6 <= 5 เท็จ → หยุดวน ออกจาก loop

ลองเล่น Loop Simulator เห็นตัวนับ i เดินทีละรอบแบบเคลื่อนไหว เข้าใจง่ายขึ้นเยอะ

while loop — วนจนกว่าเงื่อนไขจะเท็จ

while = “ตราบใดที่ยังจริง ให้ทำต่อ”

เขียนเงื่อนไขไว้ก่อน ถ้าจริงก็ทำคำสั่งในปีกกา แล้ววนกลับมาเช็กใหม่

⚠ อย่าลืม เพิ่มค่า i ในลูป ไม่งั้นเงื่อนไขจริงตลอด = วนไม่รู้จบ (infinite loop)

int i = 1; while (i <= 5) { cout << i << " "; i++; }
ผลลัพธ์: 1 2 3 4 5
for กับ while ต่างกันยังไง

เลือกใช้ให้เหมาะกับงาน

ใช้ for เมื่อ...

รู้จำนวนรอบแน่นอน เช่น พิมพ์ 1–100, วน array 5 ช่อง, ทำซ้ำ 10 ครั้ง — ตัวนับชัดเจน

ใช้ while เมื่อ...

ไม่รู้จำนวนรอบ วนจนกว่าเงื่อนไขจะเปลี่ยน เช่น “วนจนกว่าผู้ใช้พิมพ์ 0”, “วนจนเงินหมด”

ทั้งคู่วนซ้ำเหมือนกัน — ต่างที่ for เก็บ 3 ส่วน (เริ่ม/เงื่อนไข/เพิ่ม) ไว้ที่เดียว เลยเหมาะกับรอบที่นับได้

นำไปใช้จริง — หาผลรวม

บวกเลข 1 ถึง 100 ด้วยลูป

int sum = 0; for (int i = 1; i <= 100; i++) { sum = sum + i; } cout << "ผลรวม = " << sum;
sum = 0

เตรียมตัวแปรเก็บผลรวม เริ่มที่ 0

sum = sum + i

ทุกรอบ เอา i บวกสะสมเข้า sum

ผลรวม = 5050
อาเรย์ (Array) คืออะไร

ตัวแปรเดียว เก็บได้หลายค่า

แทนที่จะสร้างตัวแปร 5 ตัว (score1, score2, ...) ใช้ อาเรย์ เก็บเรียงต่อกันเป็นกล่อง ๆ — เลขกำกับกล่อง (index) เริ่มที่ 0

int score[5] = {80, 95, 70, 60, 100};

เข้าถึงสมาชิกในอาเรย์

เรียกใช้ด้วยเลข index

เขียน score[เลขช่อง] เพื่อหยิบค่านั้นมาใช้

ช่องแรกคือ score[0] ไม่ใช่ score[1] — มือใหม่พลาดตรงนี้บ่อย!

อาเรย์ 5 ช่อง index ใช้ได้ 0 ถึง 4 เท่านั้น

cout << score[0]; // 80 cout << score[2]; // 70 cout << score[4]; // 100 // เปลี่ยนค่าก็ได้ score[1] = 88; // 95 -> 88
Loop + Array = พลังที่แท้จริง

วนอ่านข้อมูลทั้งชุดด้วยลูปเดียว

int score[5] = {80, 95, 70, 60, 100}; for (int i = 0; i < 5; i++) { cout << score[i] << " "; }
i = 0; i < 5

เริ่ม index ที่ 0 วนถึง 4 (5 ช่องพอดี)

score[i]

แต่ละรอบหยิบช่องที่ i มาพิมพ์

80 95 70 60 100
เครื่องมือ & กิจกรรมวันนี้

วันนี้เราทำอะไรได้บ้าง

  • สั่งคอมทำงานซ้ำด้วย for (รู้จำนวนรอบ) และ while (วนจนเงื่อนไขเท็จ)
  • เก็บข้อมูลหลายค่าด้วย array และเข้าถึงด้วย index (เริ่มที่ 0)
  • รวม Loop + Array วนอ่าน/ประมวลผลข้อมูลทั้งชุด
  • พรุ่งนี้: เอาทุกอย่างมารวมกัน ลองโจทย์สไตล์ คอมพิวเตอร์โอลิมปิก!

การวนซ้ำ + อาเรย์ คือหัวใจของการแก้โจทย์เขียนโปรแกรมเกือบทุกข้อ — ฝึกให้คล่องวันนี้ พรุ่งนี้จะสนุกมาก
