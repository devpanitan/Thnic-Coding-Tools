อาเรย์ (Array)

ใส่ค่าในกล่อง — เข้าถึงทีละช่องด้วย index (เริ่มที่ 0) หรือกดวน for ดูทั้งชุด

โค้ดโปรแกรม

int score[5]
ผลลัพธ์ (Output)—
index เริ่มที่ 0 เสมอ — อาเรย์ 5 ช่อง ใช้ index ได้ 0 ถึง 4
นำไปต่อยอด — ลองใช้ for วนทั้งอาเรย์เพื่อ:
หาผลรวมทุกช่อง: sum += score[i];
หาคะแนนมากสุด: if(score[i]>max) max=score[i];
นับคนสอบผ่าน: if(score[i]>=50) pass++;
