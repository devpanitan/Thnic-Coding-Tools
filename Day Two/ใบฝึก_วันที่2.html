โจทย์ฝึก

1) ประกาศตัวแปรแนะนำตัว
สร้าง int เก็บอายุ และ string เก็บชื่อตัวเอง แล้ว cout พิมพ์ออกมา
2) รับค่าด้วย cin แล้วบวกกัน
เขียนโปรแกรมรับเลข 2 ตัว (a, b) แล้วพิมพ์ผลบวก
3) ตัวดำเนินการ (เติมคำตอบ)
ถ้า a = 17, b = 5 จงหา:

a / b = __________ (int)    a % b = __________     a * b = __________

4) เงื่อนไข if / else
รับคะแนน score ถ้า ≥ 50 พิมพ์ ผ่าน ไม่งั้นพิมพ์ ไม่ผ่าน
5) โจทย์รวม — เช็กอายุ
รับอายุ ถ้า ≥ 18 พิมพ์ ผู้ใหญ่ ไม่งั้นพิมพ์ เยาวชน
