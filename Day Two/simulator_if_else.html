เงื่อนไข if / else / else if

โปรแกรมเดินทีละบรรทัด ดูว่าเลือกทำทางไหน
score = score
score >= 50 ?
▼
จริง (true)
cout << "ผ่าน";
เท็จ (false)
cout << "ไม่ผ่าน";
if = ถ้า… ถ้าเงื่อนไขในวงเล็บเป็น จริง → ทำในปีกกาแรก, ถ้า เท็จ → ข้ามไปทำ else (เลือกทำอย่างใดอย่างหนึ่ง)
else if = ตรวจต่อเป็นชั้น ๆ โปรแกรมเช็คจากบนลงล่าง เจอเงื่อนไขอันแรกที่จริงก็ทำแล้วหยุด ไม่ตรวจอันที่เหลือ (เพราะฉะนั้น score 95 ได้ A ไม่ใช่ B แม้ 95 ≥ 70 ก็จริง)
