เงื่อนไข if / else

ใส่ค่าและเงื่อนไข กด “รัน” ดูว่าโปรแกรมเดินไปทางจริง (true) หรือเท็จ (false)
score = score
score >= 50 ?
▼
จริง (true)
cout << "ผ่าน";
เท็จ (false)
cout << "ไม่ผ่าน";
if = ถ้า… ถ้าเงื่อนไขในวงเล็บเป็น จริง → ทำคำสั่งในปีกกาแรก, ถ้า เท็จ → ข้ามไปทำในส่วน else แทน (โปรแกรมเลือกทำอย่างใดอย่างหนึ่งเท่านั้น)
