โครงสร้างโปรแกรม C++

ทุกโปรแกรม C++ มีโครงเหมือนกัน

จำโครงนี้ให้ขึ้นใจ — เริ่มทุกโปรแกรมจาก 4 ส่วนนี้เสมอ แล้วค่อยเขียนคำสั่งเพิ่มตรงกลางใน main()

#include <iostream> using namespace std; int main() { // เขียนคำสั่งตรงนี้ return 0; }
#include <iostream>

ขอเครื่องมือ “พิมพ์/รับ” ข้อความ

using namespace std;

ใช้ชื่อย่อ cout / cin ได้เลย

int main() { }

ตัวโปรแกรมหลัก คอมเริ่มทำงานในนี้

return 0;

บอกว่าโปรแกรมจบเรียบร้อย

Syntax — กฎการเขียน

เขียนผิดกฎนิดเดียว = error

  • ทุกคำสั่งจบด้วยเครื่องหมาย ;
  • กลุ่มคำสั่งอยู่ในวงเล็บปีกกา { }
  • ข้อความ (string) อยู่ในเครื่องหมาย " "
  • คอมเมนต์ (คำอธิบาย คอมไม่อ่าน) ใช้ // หรือ /* */

คอมพิวเตอร์ไม่ยืดหยุ่นเหมือนคน — ลืม ; หรือพิมพ์ผิดนิดเดียวก็ไม่ทำงาน แต่ไม่ต้องกลัว เพราะ error จะบอกบรรทัดที่ผิดให้เราแก้ได้เสมอ

Output — แสดงผล (cout)

cout = พิมพ์ข้อความออกหน้าจอ

ใช้ cout << ตามด้วยสิ่งที่อยากพิมพ์ ต่อกันได้หลายอันด้วย <<

endl = ขึ้นบรรทัดใหม่

พิมพ์ค่าตัวแปรก็ได้ เช่น cout << age; — ตัวแปร ไม่ต้อง ใส่ " " (ถ้าใส่จะพิมพ์คำว่า age แทนค่า)

#include <iostream> using namespace std; int main() { cout << "คะแนน = " << 90 << endl; cout << "เยี่ยมมาก!"; return 0; }
คะแนน = 90
เยี่ยมมาก!
Input — รับค่า (cin)

cin = รับค่าจากผู้ใช้

ใช้ cin >> ตามด้วยชื่อตัวแปรที่จะเก็บค่าที่พิมพ์เข้ามา

ลูกศรกลับด้านกับ cout — มองว่า “ข้อมูลไหลเข้าตัวแปร”

เคล็ดลับ: ใช้ cout บอกผู้ใช้ก่อนเสมอว่าให้กรอกอะไร แล้วค่อย cin รับค่า

#include <iostream> using namespace std; int main() { int age; cout << "กรอกอายุ: "; cin >> age; cout << "อายุของคุณคือ " << age; return 0; }
กรอกอายุ: 15 ← ผู้ใช้พิมพ์
ตัวแปร (Variable)

ตัวแปร = กล่องเก็บค่า

ประกาศแบบ: ชนิด ชื่อ = ค่า;

เปลี่ยนค่าใหม่ได้ตลอด เช่น age = 16; ค่าจะทับของเดิม

ตั้งชื่อให้สื่อความหมาย (age, score) ใช้ตัวอักษร/ตัวเลข/_ ห้ามขึ้นต้นด้วยตัวเลข

#include <iostream> using namespace std; int main() { int age = 15; string name = "แนน"; age = 16; // เปลี่ยนค่า cout << name << " อายุ " << age; return 0; }
ชนิดข้อมูล (Type)

เลือกกล่องให้ตรงกับชนิดของค่า

คอมต้องรู้ “ชนิด” ของข้อมูลก่อน เพื่อจองพื้นที่หน่วยความจำให้พอดี และรู้ว่าจะคำนวณ/แสดงผลค่านั้นอย่างไร

int

จำนวนเต็ม — 15, -3, 100

float / double

ทศนิยม — 3.14, 9.5

char

อักขระ 1 ตัว — 'A', '5'

string

ข้อความ — "สวัสดี"

bool

จริง/เท็จ — true, false

เลือกชนิดผิด ค่าจะเพี้ยน เช่น เก็บ 3.14 ใน int จะเหลือ 3

ตัวดำเนินการ (Operators)

บวก ลบ คูณ หาร และเศษ

% (mod) = เศษจากการหาร — มีประโยชน์มาก เช่น เช็คเลขคู่/คี่

ระวัง: int / int จะปัดเศษทิ้ง (7 / 2 = 3)

ลำดับการคำนวณ: * / % ทำก่อน + − (เหมือนคณิตศาสตร์) ใช้วงเล็บ ( ) บังคับลำดับได้

7 + 2 = 9 7 - 2 = 5 7 * 2 = 14 7 / 2 = 3 // ปัดเศษ 7 % 2 = 1 // เศษ
เงื่อนไข if / else

ให้โปรแกรม “ตัดสินใจ”

ถ้าเงื่อนไข จริง → ทำในปีกกาแรก, ถ้า เท็จ → ทำในส่วน else

เครื่องหมายเทียบ: > < >= <= == (เท่ากับ) != (ไม่เท่ากับ)

ข้อควรระวัง: == (เปรียบเทียบว่าเท่ากันไหม) ต่างจาก = (กำหนดค่า) — มือใหม่มักสับสน

#include <iostream> using namespace std; int main() { int score = 45; if (score >= 50) { cout << "ผ่าน"; } else { cout << "ไม่ผ่าน"; } return 0; }
ตัวอย่างรวม — โปรแกรมเช็กอายุ
#include <iostream> using namespace std; int main() { int age; cout << "กรอกอายุ: "; cin >> age; if (age >= 18) cout << "ผู้ใหญ่"; else cout << "เยาวชน"; return 0; }
int age;

สร้างตัวแปรเก็บอายุ

cin >> age;

รับค่าจากผู้ใช้

if (age >= 18)

ตรวจเงื่อนไขแล้วเลือกผลลัพธ์

รวม cin + if-else + cout
