[หลัก] = ต้องใช้   /   [เสริม] = ใช้เมื่อมีเวลา/เด็กเริ่มล้า
เปิดไฟล์ทั้งหมดได้จาก  เครื่องมือสอน_หน้ารวม.html (วันที่ 2)

────────────────────────── ช่วงเช้า ──────────────────────────

1) สไลด์ 1–2  [หลัก]  (~15 นาที)
   เปิด "สไลด์นำเสนอวันที่ 2"
   - สไลด์ 1: ไตเติล
   - สไลด์ 2: เป้าหมายวันนี้ + ทบทวน Hello World เมื่อวาน

2) สไลด์ 3–4 — โครงสร้างโปรแกรม & Syntax  [หลัก]  (~25 นาที)
   - อธิบาย #include / main / { } / ; / //
   - [เสริม] เปิด "เกมหา error" (ของวันที่ 1) ทบทวนการหา ; ที่หาย

3) สไลด์ 5–6 — Output (cout) & Input (cin)  [หลัก]  (~30 นาที)
   - cout << แสดงผล , cin >> รับค่า
   - [หลัก] เปิด online compiler (OneCompiler) ให้เด็กลองพิมพ์ cout/cin จริง
   *** พักเบรก ***

4) สไลด์ 7–8 — ตัวแปร & ชนิดข้อมูล  [หลัก]  (~40 นาที)
   - สไลด์ 7: ตัวแปร = กล่องเก็บค่า
   - สไลด์ 8: ชนิดข้อมูล int/double/char/string/bool
   - [หลัก] เปิด "Simulator ตัวแปร & ชนิดข้อมูล" ประกาศ/เปลี่ยนค่าให้ดู
   - [เสริม] เล่น "เกมจับคู่ชนิดข้อมูล" แย่งกันตอบ

────────────────────────── ช่วงบ่าย ──────────────────────────

5) สไลด์ 9 — ตัวดำเนินการ (Operators)  [หลัก]  (~30 นาที)
   - + − * / % และการหารปัดเศษของ int
   - [หลัก] เปิด "Simulator Operators" ใส่ a,b สลับ int/double ดูผลต่าง

6) สไลด์ 10 — เงื่อนไข if / else  [หลัก]  (~35 นาที)
   - โครงสร้าง if(...){ } else { } และเครื่องหมายเทียบ
   - [หลัก] เปิด "Simulator if/else" ใส่ค่า กดรัน ดูเส้นทาง true/false
   *** พักเบรก ***

7) สไลด์ 11 — ตัวอย่างรวม + ลงมือทำ  [หลัก]  (~60 นาที)
   - สไลด์ 11: โปรแกรมเช็กอายุ (รวม cin + if-else + cout)
   - ให้เด็กเขียนจริงใน OneCompiler ตามตัวอย่าง
   - แจก "ใบฝึก C++ วันที่ 2" ทำเป็นทีม (รวมโจทย์ท้าทายเลขคู่-คี่)

8) สไลด์ 12–13 — สรุป + ปิดวัน  (~15 นาที)
   - ทบทวนสิ่งที่ได้เรียน
   - เกริ่นวันที่ 3: การวนซ้ำ (Loop) และ Array

────────────────────────────────────────────────────────────
สรุปลำดับสั้น ๆ:
สไลด์1-2 → สไลด์3-4 Structure/Syntax (+เกม error) →
สไลด์5-6 cout/cin + OneCompiler → เบรก →
สไลด์7-8 + Sim ตัวแปร (+เกม Type) → [บ่าย] สไลด์9 + Sim Operators →
สไลด์10 + Sim if-else → เบรก →
สไลด์11 ตัวอย่างรวม + OneCompiler + ใบฝึก → สไลด์12-13 ปิด
