สิ่งสำคัญ: ไม่ต้องเปลี่ยนชื่อไฟล์ไทยเป็นอังกฤษ! (ลิงก์จะพัง)
            เปลี่ยนแค่ไฟล์หน้ารวมเป็น index.html ไฟล์เดียวพอ

──────────────────────────────────────────
ขั้นที่ 0 — เตรียมไฟล์ (ทำในเครื่องก่อน)
──────────────────────────────────────────
• เปลี่ยนชื่อไฟล์  เครื่องมือสอน_หน้ารวม.html  ->  index.html
  (คลิกขวา > Rename) — เพื่อให้เป็นหน้าแรกอัตโนมัติ
• ตรวจว่าจะอัปไฟล์/โฟลเดอร์เหล่านี้ครบ:
   - index.html  (หน้ารวม)
   - โฟลเดอร์ Day One\  (ต้องมี srru-logo.png + รูป slide02–slide13 + ไฟล์ .html ทุกตัว)
   - โฟลเดอร์ Day Three\  (simulator_loop.html)
  *ไฟล์ .doc / .pdf ไม่ต้องอัปก็ได้ (ไม่ใช่หน้าเว็บ) แต่อัปไปด้วยก็ไม่เสียหาย

──────────────────────────────────────────
ขั้นที่ 1 — สร้างบัญชี & repo
──────────────────────────────────────────
1. สมัคร/เข้าสู่ระบบ github.com
2. กดปุ่ม +  (มุมขวาบน) > New repository
3. ตั้งชื่อ repo เช่น  thnic-coding-tools
4. เลือก Public  >  กด Create repository

──────────────────────────────────────────
ขั้นที่ 2 — อัปไฟล์
──────────────────────────────────────────
5. ในหน้า repo กด  "uploading an existing file"
   (หรือ Add file > Upload files)
6. ลากไฟล์ index.html และโฟลเดอร์ Day One, Day Three เข้าไปวาง
   (ลากทั้งโฟลเดอร์ได้เลย GitHub จะเก็บโครงสร้างให้)
7. เลื่อนลงล่าง กด  Commit changes

──────────────────────────────────────────
ขั้นที่ 3 — เปิด GitHub Pages
──────────────────────────────────────────
8. ในหน้า repo ไปที่  Settings  (แท็บบนสุด)
9. เมนูซ้าย เลือก  Pages
10. หัวข้อ Build and deployment > Source: เลือก "Deploy from a branch"
11. Branch: เลือก  main  /  โฟลเดอร์  /(root)  >  กด Save
12. รอ 1–2 นาที รีเฟรชหน้า จะมีลิงก์ขึ้นมา:
    https://<ชื่อคุณ>.github.io/thnic-coding-tools/

เปิดลิงก์นั้น = หน้ารวมเครื่องมือ คลิกการ์ดเข้าแต่ละสื่อได้เลย
ส่งลิงก์ให้นักเรียนเปิดในมือถือ/คอมได้ทุกเครื่อง (ต้องมีเน็ต)
