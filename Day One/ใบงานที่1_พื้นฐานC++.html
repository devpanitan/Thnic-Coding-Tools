ตอนที่ 1  ความเข้าใจเบื้องต้น (เขียนอธิบายด้วยคำพูดของตัวเอง)
1) “โปรแกรมคอมพิวเตอร์” คืออะไร? ลองอธิบายสั้น ๆ
2) ภาษา C++ คืออะไร และเราเขียนภาษา C++ ไป “เพื่ออะไร”?
3) ทำไมคอมพิวเตอร์ต้องมี “คำสั่ง” ถึงจะทำงานให้เราได้?
4) ยกตัวอย่างสิ่งของหรือแอปรอบตัว ที่ทำงานด้วยโปรแกรม (อย่างน้อย 3 อย่าง)
ตอนที่ 2  คำสั่งพื้นฐานที่ได้เรียนวันนี้ (เติมคำตอบสั้น ๆ)
5) คำสั่งที่ใช้ “แสดงผลออกหน้าจอ” ในภาษา C++ คือ
6) ทุกคำสั่งในภาษา C++ ต้องปิดท้ายด้วยเครื่องหมาย
7) ข้อความที่จะพิมพ์ออกจอ ต้องอยู่ในเครื่องหมาย
8) บรรทัดแรกของโปรแกรม (เพื่อใช้ cout) ต้องเขียนว่า  #include <>
ตอนที่ 3  ลงมือทำจริง — พิมพ์โปรแกรมในเว็บ (OneCompiler) แล้วกด Run
9) ให้พิมพ์โปรแกรมต่อไปนี้ *** cout คือคำสั่งแสดงผล
#include <iostream> using namespace std; int main() { cout << "Hello World"; return 0; }
ทำการรันเพื่อ แสดงผลการทำงาน — เขียนผลที่ได้ลงในกรอบ
10) แก้บรรทัด cout ให้ขึ้นบรรทัดใหม่ด้วย endl ดังนี้
cout << "Hello" << endl << "World";
ทำการรันเพื่อ แสดงผลการทำงาน — เขียนผลที่ได้ลงในกรอบ
ตอนที่ 4  ★ ชวนคิดต่อ

12) ชวนคิด: คิดว่า “ภาษาที่คอมพิวเตอร์เข้าใจจริง ๆ ในระดับล่างสุด” ใช้ตัวเลขอะไรบ้าง? (ใบ้: มีแค่ 2 ตัว)
